Notes for AI
  
  artskart            Species occurrences and biodiversity data via Artsdatabanken API.
  brreg               Norwegian company registry: org lookup, roles, board members via Brønnøysundregistrene API.
  dnl                 Maritime navigation: harbors, lighthouses, waypoints via Den norske los.
  einnsyn             Public records, journal entries, government correspondence via eInnsyn API.
  entur               Public transport: journey planning, departures, stops via Entur APIs.
  fdk                 Master registry of Norwegian public data APIs via Fellesdatakatalog.
  fiskeridir          Fisheries and aquaculture: farms, licenses, vessels via Fiskeridirektoratet API.
  geonorge            Place names, addresses, municipalities, coordinates via Kartverket APIs.
  havforskning        Marine research: seabed, corals, pollutants, species via Havforskningsinstituttet.
  hybasapi            Bathymetry survey coverage in Norwegian waters via Kartverket OGC API.
  kartkatalog         Spatial data catalogue: find datasets, WMS/WFS services on Geonorge.
  kystverket          Maritime infrastructure: fairways, pilot stations, speed limits via Kystverket WFS.
  met-weather         Weather forecasts, nowcast, sunrise times via MET Norway API.
  miljodir            Environmental geodata: protected areas, nature types, predators via Miljødirektoratet.
  ngu                 Geological data: radon, loose deposits, seabed, groundwater via NGU.
  nibio               Agricultural and soil data: land resources, forestry, reindeer herding via NIBIO.
  niva-thredds        Water quality monitoring: river chemistry, FerryBox data via NIVA THREDDS.
  norges-bank         Exchange rates, policy rate, government bonds via Norges Bank SDMX API.
  npolar              Svalbard and Arctic geospatial data via Norwegian Polar Institute.
  nvdb                Road database: speed limits, road objects, tunnels, bridges via NVDB API.
  nve                 Energy and water: hydropower, wind, reservoirs, flood/landslide warnings via NVE API.
  nve-geo             NVE geospatial hazard data: flood zones, quick clay, glaciers, landslide maps.
  nve-warnings        Flood, landslide, and avalanche warnings from NVE/varsom.no.
  riksantikvaren      Cultural heritage: monuments, SEFRAK, Askeladden via Riksantikvaren API.
  ssb                 Statistics Norway: population, economy, demographics via StatBank and KLASS APIs.
  stormflo            Storm surge and sea level rise consequences per municipality via Kartverket.
  stortinget          Parliamentary data: representatives, committees, votes, hearings.
  trafikkdata         Traffic volumes, ÅDT, hourly counts via Trafikkdata GraphQL API.
  vannstand           Sea level, tides, water level measurements via Kartverket tide API.
  vegbilder           Road imagery: street-level photos of Norwegian roads via Vegbilder WFS/S3.
